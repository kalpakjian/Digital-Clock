## Project Overview

- Third person character movement (walk / run / jump).
- Camera following the player.
- Basic animation state machine (idle, walk, run, jump).
- Simple demo scene for testing the controller.[page:1]

## How to Run

1. Clone or download this repository.
2. Open the project in Unity Hub / Unity Editor.
3. In the Project window, open the scene:
   - `Assets/Scenes/DEMO.unity` (or whatever your main scene is called).
4. Press Play in the Editor.

## Controls

- Move: WASD / Arrow Keys
- Jump: Space
- Camera: Mouse movement (if implemented)
- Exit Play Mode: `Ctrl + P` (or the Play button)

Update these bindings according to your actual Input settings.
